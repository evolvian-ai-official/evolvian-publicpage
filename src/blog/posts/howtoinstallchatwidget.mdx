## FAQ

### Where do I find my Public Client ID?

In your Evolvian dashboard under settings. It identifies which assistant the
widget should load.

### Should I use script mode or iframe mode?

Use script mode if you want a floating chat bubble across your site. Use iframe
mode if you need the assistant in a fixed page section.

### What if the widget does not appear?

Check your Public Client ID, script placement, browser console errors, and site
security policies such as CSP.
